📱 App Overview

Athlead is a comprehensive health and habit tracking application designed specifically for Shelter Home Nutrition & Fitness Programs. The app enables caregivers to track children's meal consumption, exercise routines, and health metrics with age-appropriate portion guidelines and structured weekly meal plans.

🎯 Target Audience

  • Shelter Home Administrators
  • Caregivers & Nutrition Staff
  • Health Coordinators
  • Children (8-16 years) - monitored usage

📖 How to Use Athlead

Getting Started

  1. Install the App
    Download and install the APK on your Android device. Open the app to see the Athlead login screen with your logo.
  2. Create Account
    Tap "Create account" and fill in your details. Include your health profile (age, height, weight) for personalized recommendations.
  3. Add Students with Region
    Go to the Students tab and add children/students. Enter their age, height, and weight, and select their State/Region (Kolkata or Delhi) for region-specific meal plans.

Daily Workflow

  1. Check Dashboard
    View the daily overview showing all students' meal and exercise progress at a glance.
  2. Track Meals
    Navigate to the Meals tab. Select a student to view their daily meal plan. Mark meals as "done" when consumed.
  3. Log Exercise
    Go to the Exercise tab. View the daily workout routine. Check off completed exercises to track progress.
  4. Generate Reports
    Use the share/export feature to generate PDF reports of weekly meal plans and exercise summaries.

Key Tips

  • Meal Tracking: Tap on any meal to see the detailed dish combination and nutritional information
  • Portion Guidelines: The app automatically adjusts portions based on each student's age group
  • Regional Meal Plans: View Kolkata or Delhi specific meal plans in the Profile tab under "Country Meal Plans"
  • Student Region Assignment: Assign students to a region during onboarding to see region-specific meal recommendations
  • Weekly View: Use the calendar to navigate between different days and view historical data
  • Multiple Students: Add all children in your care to track them from a single dashboard
  • Offline Support: Data syncs automatically when internet connection is available

🚀 Building from Source

Prerequisites

  • Node.js 18+ and npm
  • Android Studio (for Android SDK)
  • Java Development Kit (JDK) 17+
  • Expo CLI

Installation Steps

# 1. Clone the repository git clone cd tracker-app # 2. Install dependencies npm install # 3. Start development server npx expo start # 4. Run on Android device/emulator npx expo run:android

Building Production APK

# Method 1: Local Build export ANDROID_HOME=$HOME/Android/Sdk export PATH=$PATH:$ANDROID_HOME/cmdline-tools/latest/bin npx expo prebuild --platform android cd android ./gradlew assembleRelease # APK will be at: # android/app/build/outputs/apk/release/app-release.apk # Method 2: EAS Cloud Build npx eas build --platform android --profile production

🔧 Configuration

Firebase Setup

The app uses Firebase Realtime Database for data persistence. Configure your Firebase project in app.json:

{ "expo": { "extra": { "firebaseApiKey": "YOUR_API_KEY", "firebaseAuthDomain": "your-project.firebaseapp.com", "firebaseProjectId": "your-project", "firebaseDatabaseUrl": "https://your-project-default-rtdb.firebaseio.com" } } }

Environment Variables

Create a .env file for sensitive configuration (not included in version control):

FIREBASE_API_KEY=your_api_key FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com FIREBASE_PROJECT_ID=your_project FIREBASE_STORAGE_BUCKET=your_project.appspot.com FIREBASE_MESSAGING_SENDER_ID=your_sender_id FIREBASE_APP_ID=your_app_id FIREBASE_DATABASE_URL=https://your_project-default-rtdb.firebaseio.com
